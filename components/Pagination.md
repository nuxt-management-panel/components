# Pagination

### sample

```html

<div>
    <Pagination/>
</div>
```

### props

> - totalItems:Number

> - itemsPerPage:Number 
>    

> - primary
>
> *Adds the primary color to the active page*
> 

> - rounded
> 
> *Makes the active page rounded*
> 

>  - out-lined
> 
> *Changes the design of the active page to out lined*
> 

###Emits

>page
> 
> *Returns the requested page number*
